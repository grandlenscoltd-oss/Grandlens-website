# Grandlens Company — Official Website

This repository contains the source code for the official website of **Grandlens Company**, Nairobi's premier printing, branding, and graphics design studio. The site is a high-performance, single-page application designed to convert visitors into clients through a modern, "undeniable" visual identity.

## 🔗 Live Production
The website is currently deployed and live at:
**https://grandlens-company.netlify.app/**


## 🛠 Tech Stack
* **Frontend:** Semantic HTML5, CSS3 (Custom Properties & Flex/Grid), Vanilla JavaScript (ES6+).
* **Typography:** Syne (Headings) and DM Sans (Body) via Google Fonts.
* **Animations:** Intersection Observer API for scroll-driven "fade-up" effects.
* **Deployment:** Netlify.

## 🚀 Deployment & Maintenance
As a solo maintainer, the workflow is streamlined for quick updates:

1. **Local Development:** Open `index.html` in any modern browser to preview changes.
2. **Production Updates:** Push changes to the `main` branch. Netlify is configured to automatically trigger a build and deploy the latest version.
3. **Lead Management:** The "Get a Quote" form uses a custom JavaScript handler (`handleSubmit`) to manage user requests. Ensure the email endpoint in the form script remains updated to `grandlenscoltd@gmail.com`.

## 🎨 Design Guidelines
To maintain the brand's premium aesthetic, follow these style constraints:

Primary Red: #C8102E.

Background: High-contrast Dark/Black (#0f0f0f).

Imagery: Focus on high-quality visual representation for services like Large Format printing, Screen Printing, and Corporate Gifts.
